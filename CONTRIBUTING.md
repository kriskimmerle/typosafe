# Contributing to typosafe

Thanks for your interest in contributing!

## Quick Start

1. Fork the repo
2. Clone your fork: `git clone https://github.com/YOUR-USERNAME/typosafe`
3. Install: `pip install -e .`
4. Make changes
5. Test: Run `typosafe` on sample requirements files
6. Push and open a PR

## What to Contribute

- New typosquatting detection patterns
- Performance improvements
- Bug fixes
- Documentation improvements
- Additional package manager support (npm, cargo, etc.)

## Code Style

- Python 3.7+ compatible
- Keep dependencies minimal
- Clear variable names
- Comments for non-obvious logic

## Reporting Issues

Open an issue with:
- What you tried
- What you expected
- What happened instead
- Sample requirements file (if applicable)

## Ideas for Contributions

- Add fuzzy matching algorithms
- Support for poetry.lock, Pipfile.lock
- Integration with CI/CD tools
- Database of known typosquatting packages
- Levenshtein distance thresholds tuning
